# 🚀 py-logic-lab

> Exercícios de lógica de programação implementados em Python puro — estilo Rocketseat.

---

## 📋 Sobre o projeto

O **py-logic-lab** é um projeto educativo para iniciantes em programação. Ele reúne 8 exercícios clássicos de lógica em um menu interativo no terminal, cobrindo os fundamentos essenciais do Python: entrada de dados, condicionais, loops, funções e listas.

Ideal para quem está começando e quer ver a teoria ganhar vida no código. ✨

---

## ⚙️ Funcionalidades

| # | Exercício | Conceito praticado |
|---|-----------|-------------------|
| 1 | Somar dois números | `input()` + operadores matemáticos |
| 2 | Par ou Ímpar | `if/else` + operador `%` |
| 3 | Contagem regressiva | `while` + decremento |
| 4 | Múltiplos de 3 até 30 | `for` + `range()` + listas |
| 5 | Calcular média de 3 notas | operadores + `float` |
| 6 | Verificar aprovação | `if/elif/else` encadeado |
| 7 | Maior número entre dois | condicional simples |
| 8 | Positivo, negativo ou zero | condicional múltipla |

---

## 🛠️ Tecnologias

- **Python 3** — sem dependências externas
- Terminal (qualquer sistema operacional)

---

## ▶️ Como executar

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/py-logic-lab.git

# Acesse a pasta
cd py-logic-lab

# Execute o projeto
python main.py
```

> Certifique-se de ter o **Python 3** instalado: `python --version`

## 🧠 O que você aprende aqui

- Criar **funções** com responsabilidade única
- Usar **`if/elif/else`** para tomar decisões
- Controlar fluxos com **`while`** e **`for`**
- Coletar dados do usuário com **`input()`**
- Tratar erros simples com **`try/except`**
- Organizar código em **módulos separados**
- Trabalhar com **listas** para armazenar resultados
